# Contributing to Awesome MLX

Thanks for helping grow the MLX ecosystem!

## Adding a Project

**Option 1: Open an issue** (easiest)

[Use the submission form](https://github.com/raullenchai/awesome-mlx/issues/new?template=submit_project.yml) — just fill in the fields and we'll add it.

**Option 2: Submit a PR**

1. Fork this repo
2. Add your project to the correct category in `README.md`
3. Use this format:
   ```
   - 🐍 [name](https://github.com/owner/repo) — One-line description. ![](https://img.shields.io/github/stars/owner/repo?style=flat-square)
   ```
4. Sort by stars (descending) within the section
5. Open a PR

## Guidelines

- **All MLX projects welcome** — no minimum star count
- **One-line descriptions** — keep it concise
- **Language emoji**: 🐍 Python · 🦅 Swift · 🟨 JS/TS · 🦀 Rust · 🐹 Go
- **Pick the right category** — if unsure, mention it in the PR and we'll help

## Reporting Issues

Found a broken link? Project moved or archived? Open an issue or PR to fix it.
